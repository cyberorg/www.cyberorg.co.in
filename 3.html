Novell Authorized Training

Novell courses are synonymous with quality, in-depth concept coverage and clarity. Being the pioneer of technology training, the courses offered are tailored not only to the today's requirement but prepare you for the Open future.

Certification Oriented Courses

Novell pioneered the IT on-line certification globally. Currently Novell is offering Novell certifications and Suse certifications which are recognised and acclaimed worldwide.

1.1 Certified Novell Administrator (CNA)

CNAs provide companies direct support for software users in various work environments including professional offices and small businesses, work groups or departments, and corporate information services (IS).

Foundations of Novell Networking (Course 3001)

This course includes fundamental information about NetWare 6. In addition to exploring core concepts, this course explains NetWare 6 implementation and teaches you how to use its convenient management tools. Students also learn how to use other key components of Novell networks, including GroupWise Messaging and Desktops Management with ZENworks.

1.2 Certified Novell Engineer (CNE)

Foundations of Novell Networking (Course 3001)

Novell Network Management: NetWare 6 (Course 3004):

This course provides students with the knowledge and skills to perform network management tasks that are the most common yet critical to administrators of LANs and WANs. Emphasis is on the use of network management utilities of NetWare 6, deployment of NetWare 6 features and troubleshooting skills that a student in an entry-level network administration position must have to manage the daily network operations of an organization.

Advanced Novell Network Management: NetWare 6 (Course 3005):

This is a self-study and instructor-led course that offers advanced techniques for managing and troubleshooting networks that include NetWare ® 6. Specifically, Advanced Novell Network Management provides extensive knowledge in the following key areas: Upgrading to, and managing, NetWare 6 in heterogeneous networking environments, Advanced network storage management, High-availability services with Novell Cluster Services ™ , Finding internal security breaches with Novell Advanced Audit Services.

Novell eDirectory Design & Implementation (Course 575):

In this course, students will learn to design and implement Novell eDirectory trees and related components in any type of organization for different types of organizational goals using different types of network operating systems. Students familiar with a variety of operating systems will benefit most from this course. Students will work with NetWare, Linux, and Windows 2000 servers using Novell eDirectory. This course is core to the CNA , CDE and CNI tracks.

Desktop Management with ZENworks for Desktops 4 (Course 3006):

This is an educational offering designed for students who have little or no experience with the ZENworks product family. This version includes enhanced lab instructions. Specifically, this course is designed for students who are semi-experienced with networking, although not necessarily experienced with NetWare ®.Students should also have a conceptual knowledge of networking protocols, including TCP/IP, Networking Hardware, Networking Topologies and OSI Model.

1.3 Novell Certified Linux Professional (NCLP)


Click Image to enlarge.

The Novell Certified Linux Professional (Novell CLP) is for people interested in being Linux administrators. Skills demonstrated by someone holding a Novell CLP certification include:installing Linux servers into a network environment, managing users and groups, troubleshooting the SUSE LINUX file system, managing and compiling the Linux kernel, and troubleshooting network processes and services--just to name a few. As with all Novell certifications, course work is never required. One need only pass a Novell Practicum (050-689)in order to achieve the certification. The Novell authorized training materials for this certification are designed to introduce students to basic Linux concepts and progress them through the tasks of managing a Linux network environment.

Course Descriptions:

A. Linux Fundamentals (Course 3036)

Introduces open source standards and common knowledge and skills needed in all Linux distributions. Attendees gain the essential skills required to log in to a multiuser Linux environment, navigate the SUSE Linux file system and manipulate files, work within shells and execute shell script commands, control processes running on the SUSE LINUX Server, and more.

B. Linux Administration (Course 3037)

Teaches administrative skills common to an entry-level administrator or help desk technician in an enterprise environment. Attendees learn to conduct common, entry level system administration tasks, such as performing a Linux server installation into a network environment, granting user and group permissions, managing users and groups, managing software applications, troubleshooting the SUSE LINUX file system, managing printers, configuring the network, and more.

C. Linux Advance Administration (Course 3038)

Presents advanced administrative skills common to mid-to senior-level administrators in an enterprise environment. Attendees learn how to apply security to network users and resources, manage and compile the Linux kernel, manage network clients, implement an FTP server, manage and configure Web servers, use OpenLDAP to manage network data, manage mail and messaging services, and troubleshoot network processes and services..

The Novell Practicum

Novell Practicums are scenario based exams designed to simulate real-life situations one might encounter in the normal operation of a network. Each practicum is designed to test specific skills & knowledge relevant to the potential job role the certification applies to. Example:The Novell Practicum for a Novell CLP might require one to successfully install a SUSE LINUX server into a heterogeneous network. Practicums are graded on a pass/fail basis.

1.4 Fundamentals of Novell Networking

This course teaches students with basic knowledge about the layers of the OSI model and the features of the components that make up a network. Upon completion of this course, students will have the skills needed to install, configure, and troubleshoot basic networking hardware peripherals and protocols. Based on CompTIA's Network+ exam, this course covers topics on media and topologies; protocols and standards; network implementation and support. Please note, this course is the suggested pre-requisite for Foundations of Novell ® Networking:NetWare ® 6 (Course 3001).

1.5 Novell Linux Desktop

This course will help the user to make the most of what is likely his first experience with an Open Source desktop, Novell has developed a comprehensive course to help the Linux user to get up and running smoothly and effectively on the Novell Linux Desktop.
