Server Solutions

Mail

Postfix running on Linux is considered the best mail server solution today. Postfix with additional services like Dovecot/Cyrus, spamassasin, MailScanner-ClamAV antivirus and next generation Horde webmail client makes it an ideal choice to replace MS Exchange. Apart from this you can use LDAP or PostgreSQL server to store preferences, Addressbooks, Calendar etc. You can also use popular POP3 or IMAP clients, making the migration smoother.

If you are looking for something fun and easy to use check out Hula server which is a Novell opensource project.

Anti-Virus Gateway

A secure Squid proxy server scans all the data coming into your network for harmful viruses and acts as a shield against any harmful codes that can infect user computers. Proxy server combined with mail server with antivirus and most secure IPtables based firewall guards against any potential threats.

LTSP

Linux Terminal Server Project is an addon to Linux server making it capable of serving a number of thin clients. A thin client is a typical low configuration, diskless PC such as 486, ,AMD-K6, P1, P2 or P3 with pxe capable ethernet card.

Why throw away your old hardware when they can work as fast as any latest specification hardware?

An AMD 3000+ or a P4 equal with 2 GB of RAM can serve over 50 thin clients easily.
